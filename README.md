# Projeto Conceitual de Banco de Dados – E-commerce

Este projeto tem como objetivo a modelagem conceitual de um banco de dados para um sistema de e-commerce, refinando o modelo inicial proposto pela DIO.

## Requisitos Atendidos

### Cliente PF e PJ
Foi utilizada uma especialização da entidade Cliente em Cliente_PF e Cliente_PJ, garantindo que uma conta possa ser apenas pessoa física ou pessoa jurídica, nunca ambas.

### Pagamento
O sistema permite o cadastro de múltiplas formas de pagamento por cliente, além da associação das formas de pagamento aos pedidos realizados.

### Entrega
A entidade Entrega contempla informações como status da entrega e código de rastreio, permitindo o acompanhamento do pedido até a entrega final.

## Tecnologias
- Modelagem Conceitual (DER)
- Banco de Dados Relacional
